# meteor-coininfo

JS component for crypto currency specific information such as version numbers, DNS seeds, etc.
Ported from npm, bundled with browserify.


### Official documentation:

[http://cryptocoinjs.com/modules/currency/coininfo/](http://cryptocoinjs.com/modules/currency/coininfo/)

####Example
Exposes a camel-cased function (`CoinInfo`) to the client.

`CoinInfo('BTC')` for example will give you an object with relevant information for Bitcoin

Please check the official documentation for supported crypto currencies.
